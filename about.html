About

What is StockBot?

StockBot is a web application designed for user-friendly stock analysis. This is done through quantitative analysis.

Who are you?

My name is John. I am a hobbyist programmer and retail investor. I have always had a passion for programming and numbers. Quantitative analysis of stocks is a beautiful money-making science to me. This project represents that.

What language is StockBot built in?

At the moment StockBot is built in Vanilla Javascript. Charts are displayed using the third-party library chartjs.

What are you currently working on for StockBot?

This is a home project as of right now. So, I work on this in my spare time. If you'd like to see what I'm currently working on, you can look at my Trello board.
